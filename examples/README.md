# Examples Directory

This directory contains various examples demonstrating different use cases for Fetch.AI UAgents.

## Available Examples

### Basic Examples (Root Directory)
- `../agent.py` - Basic agent with essential features
- `../agent_advanced.py` - Advanced features with protocols and storage
- `../agent_communication.py` - Two agents communicating with each other
- `../example_usage.py` - Custom task processor example

### Additional Examples (Coming Soon)
- Weather service agent
- Price oracle agent
- Task scheduler agent
- API integration agent
- Multi-agent coordination

## Running Examples

Each example can be run independently:

```bash
# From the root directory
python agent.py

# Or from specific example
python examples/specific_example.py
```

## Creating Your Own Example

1. Copy an existing example as a template
2. Modify the agent name and seed
3. Update the message models
4. Implement your business logic
5. Test locally before deploying

## Tips

- Always use unique agent names and seeds
- Use different ports for each agent
- Test agent communication locally first
- Read the comments in each example
- Check the main README for more information
